# Política de Segurança

## Versões Suportadas

Use esta seção para informar as pessoas sobre quais versões do seu projeto estão
sendo suportadas atualmente com atualizações de segurança.

| Versão | Suportado          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

## Relatando uma Vulnerabilidade

Use esta seção para dizer às pessoas como relatar uma vulnerabilidade.

Diga a elas onde devem ir, com que frequência podem esperar obter uma atualização sobre uma
vulnerabilidade relatada, o que esperar se a vulnerabilidade for aceita ou
recusada, etc.
