# 安全策略

## 受支持的版本

使用此部分告诉人们目前支持哪些安全更新的您的项目版本。

| 版本 | 是否支持          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

## 报告漏洞

使用此部分告诉人们如何报告漏洞。

告诉他们去哪里，他们多久可以得到关于已报告漏洞的更新，如果漏洞被接受或拒绝会发生什么等等。
