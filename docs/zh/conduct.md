# 行为准则

## 对于贡献者
1.  **尊重架构：** 除非绝对必要，否则不要引入繁重的框架（Retrofit、Hilt）。保持精简。
2.  **尊重用户：** 未经明确的选择加入同意，请勿添加跟踪、分析或遥测。
3.  **尊重设备：** 此应用程序运行会发热。优化您的循环。不要无缘无故地耗尽用户的电池。

## 对于用户（“街头”规则）
* **合法：** 我们不容忍破坏公物。确保您有权限在您正在扫描的墙壁上作画。
* **安全：** 请勿在靠近交通的地方步行时使用 AR 模式。“置信度引擎”令人着迷，但公共汽车是真实的。
