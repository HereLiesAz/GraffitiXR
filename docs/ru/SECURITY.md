# Политика безопасности

Мы серьезно относимся к безопасности приложения GraffitiXR и данных наших пользователей. В этом документе описывается наш подход к безопасности и то, как сообщать об уязвимостях.

## Поддерживаемые версии

В настоящее время обновления безопасности поддерживаются только для последней версии. Пожалуйста, убедитесь, что вы используете последнюю версию приложения.

| Версия | Поддерживается |
| ------ | -------------- |
| Последняя | Да             |
| Старые   | Нет            |

## Отчет об уязвимостях

Если вы обнаружили уязвимость в безопасности GraffitiXR, пожалуйста, не создавайте публичную проблему. Вместо этого, пожалуйста, сообщите об этом ответственно.

Вы можете сообщить об уязвимости, отправив письмо на наш контактный адрес (будет добавлен позже).

Пожалуйста, включите следующую информацию в ваш отчет:
* Подробное описание уязвимости.
* Шаги для воспроизведения проблемы.
* Любое возможное влияние уязвимости.
* Любая дополнительная информация, которая может помочь нам быстро понять и решить проблему.

Мы ценим ваши усилия по обеспечению безопасности GraffitiXR и будем работать с вами над устранением проблемы.
