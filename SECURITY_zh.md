# 安全策略

## 受支持的版本

目前支持以下版本的 GraffitiXR 提供安全更新。

| 版本    | 是否支持           |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## 报告漏洞

如果您在 GraffitiXR 中发现安全漏洞，请在 GitHub 上发起 issue，或通过 GitHub 安全公告功能私下与我们联系。我们将尽快审核并回复报告，通常在 48 小时内。

一旦漏洞被审核和接受，我们将协调时间表发布补丁。我们非常感谢您为帮助保持 GraffitiXR 安全所做的努力！
