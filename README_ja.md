# GraffitiXR

GraffitiXR はストリートアーティストのためのAndroidアプリです。カメラビューに画像をオーバーレイして仮想的にトレースできるアプリはたくさんありますが、スマートフォンに保存したスケッチに基づいて壁画を描くとき、三脚を使うのは作業の妨げになります。私たちはあちこち動き回るからです。私はスマートフォンをポケットに入れます。ARを使用して画像を安定して1つの場所に保持するアプリでさえ、ポケットの真っ暗闇には対処できません。

そこで、（知る人ぞ知る）グリッド法を再利用して、より良いものを作りました。私は常に、「これらの特定の落書きを永続的なアンカーのように保存して、オーバーレイが常に正しい場所に表示されるようにできないのはなぜだろう？」と考えていました。

それで、今ではこれらの落書きがその役割を果たします。

クラウドの助けを借りずにAndroid上で動作するカスタムのガウシアンスプラッティングエンジンを発明しなければなりませんでした--なぜなら、グラフィティは違法だからです。

そして、それに続いて、私がTeleological Slamと呼ぶものを追加しました--結果がどのようになるかは分かっているため、OpenCVを使用して進捗状況を調べます。つまり、進むにつれて、オーバーレイが壁にしっかりと貼り付きます。これがないと、絵自体でそのマークを覆ってしまい、進むにつれてアプリの精度が低下します。これこそが、このような他のアプリが本当に失敗する部分です。

また、画像トレース用の非AR画像オーバーレイ機能も含まれています。他のアプリと同じように、そのような使い方が好きな人のためです。さらに、モックアップモードもあります。壁の写真を撮って、素早いモックアップのためのツールを使います。そして、証明するものは何もなく、ただ何かを紙に完璧にコピーしたいだけなら、トレースモードでスマートフォンをライトボックスとして使用できます。画面を明るくしたままにし、画像を所定の位置にロックし、完了するまですべてのタッチをブロックします。

さらに、マルチレイヤーのグラフィック作成をサポートする、関連するデザインツールのまともなスイートもあります。まだまだ続けられますが、もう十分話したと思います。

**GraffitiXR**は、ストリートアーティスト向けのオフラインAndroidアプリです。信頼度に基づくボクセルマッピングシステムを使用して、ARで壁に画像を投影します。

## 主な機能
*   **オフラインファースト:** クラウドへの依存はありません。データは収集されません。
*   **ガウシアンスプラッティングAR:** ガウシアンスプラットを使用した正確な壁の投影。
*   **Teleological Slam:** OpenCVを使用した進行状況に基づくトラッキング。
*   **モックアップモード:** 実際の壁の写真でデザインをプレビュー。
*   **トレースモード:** デバイスをロック可能なライトボックスに変えます。
*   **マルチレイヤーデザイン:** 包括的な画像調整およびブレンドツール。
*   **ステンシルジェネレーター:** 画像を印刷可能なマルチレイヤーのステンシルPDFに変換します。

## アーキテクチャとテクノロジー
アプリは堅牢なスタックで構築されています：
*   **UI/アプリ:** Jetpack Composeを備えた最新のKotlin。
*   **AR/トラッキング:** デプスAPIとカスタムシェーダーを備えたGoogle ARCore。
*   **レンダリング:** JNIを介したカスタムOpenGL ES 3.0。
*   **コンピュータビジョン:** 特徴抽出とマッピングのためのOpenCV。

## ビルド手順
1.  リポジトリをクローンします。
2.  Android Studioで開きます。
3.  ビルドしてデバイスにデプロイします。
